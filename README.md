# COMPUTATIONAL-VISION
---
Esse repositório apresenta uma implementação do modelo de detecção de objetos YOLOv3. 

Temos dois links principais, um que mostra a implementação completa da arquitetura do modelo YOLOv3,

e onde o código pode ser executado e os resultados do modelo de detecção de objetos podem ser vistos.

Esse código não é de minha autoria e está com duas funções essenciais faltantes para o funcionamento do modelo, a IoU e a non-max suppression.

O segundo link mostra a implementação feita por mim dessas duas funções. Que são chamadas no código principal, da arquitetura completa.

## Considerações sobre o projeto

Esse projeto ilustra o poder das redes neurais convolucionais na resolução de problemas de visão computacional, aplicando um modelo pré-treinado podemos 
observar a detecção de objetos em cenas complexas, e no código podemos também mudar os parâmetros de non-max suppression e de confiança para ver como eles alteram os resultados.
